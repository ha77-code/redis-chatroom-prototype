# Redis Chatroom Prototype

## 项目简介
本项目是一个基于 Redis 中间件/数据库技术的网络聊天室原型系统，面向软件工程课程设计与原型演示场景。

系统包含普通用户与管理员两类角色：

- 普通用户支持注册、登录、修改个人信息、加入聊天室、退出聊天室、发送消息、接收消息、查看在线用户和历史聊天记录。
- 管理员除具备普通用户全部能力外，还支持禁言用户、踢出用户、删除不当聊天记录和聊天室管理。

本项目默认采用 `Java Web + Redis` 的技术路线，消息收发采用 `HTTP 轮询` 方式实现。

## 目录建议
建议按以下分层组织代码：

```text
src/main/java/com/chatroom
├─ controller
├─ service
├─ repository
├─ entity
├─ dto
├─ util
└─ store
```

## 核心功能
- 用户注册与登录
- 用户资料修改
- 加入/退出聊天室
- 发送消息与轮询接收消息
- 查看在线用户列表
- 查看历史聊天记录
- 管理员禁言用户
- 管理员踢出用户
- 管理员删除消息

## Redis 主要用途
- 保存用户 Session
- 保存在线用户状态
- 保存聊天室消息列表
- 保存禁言状态
- 保存踢出状态
- 保存聊天室基础信息

## 建议开发顺序
1. 完成实体类和 DTO
2. 完成 Redis 数据访问层
3. 完成用户与认证模块
4. 完成聊天室消息模块
5. 完成管理员功能模块
6. 完成测试与联调
